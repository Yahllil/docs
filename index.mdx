# Welcome to Crust AI\! ✨

Transform your **Airtable** data into powerful, **interactive mini-applications** – we call them **Snippets**. Share secure, live data views with clients, partners, or team members without complex coding or granting full base access.

This documentation will guide you from basic concepts to building sophisticated, interconnected applications.
